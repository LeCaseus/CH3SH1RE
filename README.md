# CheshireOS

Hey! This is my first repo ever. I built this local AI assistant just for fun as a side project.

## What you need to run it

- AMD Ryzen 5 5500U @2.1 Ghz (or equivalent)
- 16GB RAM
- NVIDIA RTX 3050 laptop GPU with 4GB VRAM (or equivalent)

## How it works

It runs on [llama.cpp](https://github.com/ggerganov/llama.cpp) server and uses the Qwen2.5 7B instruct quantized model (q5_k_m).

My goal is to give it a web search, vision, and RAG capabilities :)
