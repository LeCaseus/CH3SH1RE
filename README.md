# CheshireOS

Hello! This is my first repo ever. I built this local AI assistant just for fun as a side project.
This is my first project that will be a part of a bigger portfolio as I learn more about programming.

## What you need to run it (I think)

- AMD Ryzen 5 5500U @2.1 Ghz (or equivalent)
- 16GB RAM
- NVIDIA RTX 3050 laptop GPU with 4GB VRAM (or equivalent)

## Things you need to know

It runs on [llama.cpp](https://github.com/ggerganov/llama.cpp) server and uses the Qwen2.5 7B instruct quantized model (q5_k_m).

My goal is to give it a web search, vision, and RAG capabilities :)

Its not much right now because I am still learning a bunch of stuff about programming but I'll work on this project as I go.
